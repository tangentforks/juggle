Explicit Arguments x.  y.  

The names x. and y. denote the left and right arguments in an explicit definition. See the names m. n. u. v. and Explicit Definition (:).
 

For example:
info=: 3 : 0
 (3!:0 y.);(#$y.);($y.);5!:5 <'y.'
  :
 (info x.) ,&< (info y.)
)
