Support and Questions

Check the latest information at www.jsoftware.com.

Check the online manuals and help files available through the J Help menu.

The J forum www.jsoftware.com/j_forum.htm is the best place for general questions on the J language.

Send email inquiries to:

general info@jsoftware.com
sales sales@jsoftware.com
technical tech@jsoftware.com

When reporting problems, please include the following information:

1) Is the problem reproducible? If so, how?
2) What version of Windows are you running?
3) What version of J are you running? About J from the Help menu has this information.
