Standard profile

Some examples assume that your system runs the standard profile when it is started. This configures your system and makes some standard utilities available.

Enter CR (capital letter C followed by capital letter R) into the ijx window as a quick check on whether the profile has been run.

   CR

If the result is a blank line, then profile has been run and you can skip the rest of this section.

If instead you see:

   CR
³value error

you will have to change your J icon to run the standard profile

.

The standard profile is in file system\extras\config\profile.ijs in your J directory. You need to edit the J icon properties to indicate that this file should be run when J starts.

On the Macintosh you double-click the J icon labeled profile.ijs to start J with the standard profile.
