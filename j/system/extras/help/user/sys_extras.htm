system\extras

This directory is used for system facilities, such as labs, configuration and help files. The layout is:

extras\config

configuration files

extras\help

help files

extras\java

Java interface support

extras\labs

labs

extras\migrate

migration aids

extras\order

order form

extras\template

template files

extras\util

utilities
