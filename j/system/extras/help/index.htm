J Help System - version date: 2001-5-11

Be sure to see Release Highlights (Windows) (Unix).

There are 5 books (User, Primer, Phrases, Dictionary, Release). Every page has standard jumps:
   >> next, << previous, NDX index
   Usr Pri Phr Dic and Rel to that contents
   Voc to Vocabulary, !: to Foreigns, wd to WD reference
   Help jumps here and the current book is at the end

Additional publications (no standard jumps or index):
Computers and Mathematical Notation - Kenneth E. Iverson
Learning J - Roger Stokes

Practice navigating with the jump line and the browser back and forward to get your bearings. Then come back here for hints and information on the index.

F1 is the fast way to Help. Launch one or two and use the taskbar to move between them and J.

Most browsers allow you to change font size and color. If your browser supports style sheets, you can edit jdoc.css to customize.

The index contains all 'useful' words from the books as well as from the labs and standard library scripts.

Book references jump to the page. Use your browser search on the page and back to return to the index.

Lab references are the Lab name. Use Studio|Labs in J to run the lab.

Script references are the short name. Short names are mapped to filenames with noun PUBLIC_j_. Use the short name to directly access the file. For example:
   open 'winlib'      NB. open file
   load 'scriptdoc'   NB. scriptdoc utility
   scriptdoc 'winlib' NB. get doc from file

Scripts with no short name are listed by file name. For brevity the path and .ijs are not included. Two dots separate short names from file names.

Use Edit|Find in Files to search the book html files, scripts, and labs.
